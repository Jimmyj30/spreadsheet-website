Help: click on a cell and drag the blue square on the bottom right of the cell downwards in order to add new rows (identical to Excel autofill).

Help: right click on a cell to display a context menu for a list of possible interactions with the cell. You can also toggle a dropdown menu for each column in the table by clicking on the button in each table column header.

Help: press control-z or command-z to undo recent changes to the spreadsheets.

Note: the minimum number of rows for this data table is 5.

Note: creating a processed data table for the first time may take ~30 seconds to complete as the API that processes the data tables needs to take some time to start up.
